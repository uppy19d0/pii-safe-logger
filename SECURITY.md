# Security Policy

## Supported Versions

Security fixes are provided for the latest published version of
`pii-safe-logger`.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately through the security advisory
feature of the public source repository linked from `package.json`.

If the repository has not been published yet, contact the npm package maintainer
through the package page and avoid public disclosure until a fix is available.

Expected response:

- acknowledge the report within 72 hours
- validate impact and affected versions
- publish a patched release when the issue is confirmed
- credit the reporter when requested and appropriate
